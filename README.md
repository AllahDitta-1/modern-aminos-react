# Moderna Minos React

A React and Vite implementation of the Moderna Minos landing experience, focused on responsive layout, modern UI presentation, and clean route-based structure.

## Features

- Responsive landing flow built with React Router.
- Separate pages for the homepage, account page, and contact page.
- Modern UI styling designed for a polished marketing presentation.
- Good foundation for extending a brand landing page into a fuller product website.

## Tech Stack

- React
- Vite
- React Router DOM
- Tailwind CSS
- Lucide React
